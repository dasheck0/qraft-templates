# Testing Pipeline

This command runs the complete testing pipeline for the project.

## Usage

To run the complete testing pipeline, just type:

1. Run npm typecheck
2. Run npm lint
3. Run npm test
4. Report any failures
5. Fix any failures
6. Repeat until all tests pass
7. Report success

## What This Command Does

1. Runs `npm typecheck` to check for type errors
2. Runs `npm lint` to check for linting errors
3. Runs `npm test` to run the tests
4. Reports any failures
